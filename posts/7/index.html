酷吧，虽然咱代码不会写，但气势绝对不能输，逼格总是要先装起来的，装着装着，说不定，就会写代码了呢！！！（大误 😄）

接下来就教你们如何装逼

注意，本文许多网站需要科学上网才能使用，请自备科学上网工具

0.准备工作

首先需要新建一个和 Github 用户名相同的仓库
记得勾选Add a README file这个选项接下来在自己电脑上创建一个.md后缀的文件，名称随意，并用支持Markdown语法的编辑器编辑，当然嫌麻烦可以用https://arya.pages.dev/或者其他你喜欢的在线编辑器编辑

1.带有打字特效的文字

打字特效生成网站：Readme Typing SVG - Demo Site

里面有很多自定义选项，大家可以根据自己喜好进行调整调整完以后点击Markdown下的Copy To Clipboard，然后粘贴进刚才的.md编辑器里

2.社交关注者数目小牌子

用 Substats 配合shields.io制作动态小牌子

创建动态图标方法：点击进入这个网页 shields.io dynamic-badge

data type选择json
label写上数据牌左侧你想展示的文字，比如我想展示的是GitHub
data url，找到自己想展示的API URL，比如我的GitHub URL就是：https://api.spencerwoo.com/substats/?source=github&queryKey=fjqz177。一般都只需要修改source后面的目标服务器以及queryKey后面的请求数据标签，前者一般都是平台名，后者一般都是用户名
query填：$.data.totalSubs
color填写一个十六进制的颜色代码，前面不加#
剩下两个分别是数据牌右侧展示数据的前缀和后缀，前缀可以不填，后缀可选填followers


然后点击Make Badge即可生成以如下格式填入刚才生成的链接，并把它粘贴进.md编辑器内

效果如下

大家可以查阅Substats API的文档：Home | Substats Docs (spencerwoo.com)看看支持哪些平台以及source后面的目标服务器的关键字，举一反三即可生成你想要的小牌子给大家附上我自己带图标的小牌子

只需将对应title后面的href里的链接替换成自己的，然后把src后面的链接里的fjqz177或者对应的id号改成自己的就可以了（queryKey%3D后面的那一串）

图标是使用 base64 将 svg 进行转码的，所以很长~~
